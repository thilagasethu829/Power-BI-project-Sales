# Thilaga Power-BI-project-Sales Analysis Report
Sales Analysis Dashboard – Power BI
This interactive Power BI dashboard provides a comprehensive analysis of sales performance, helping users monitor key business metrics and make data-driven decisions. The report visualizes total sales, profit, sales growth, product performance, and regional trends through dynamic charts, slicers, and KPI cards.

Key features include:

- Sales and profit tracking
- Month-over-month sales growth analysis
- Product and category performance insights
- Regional sales comparison
- Interactive filters and slicers for deeper exploration
- Professional and user-friendly dashboard design

Built using Power BI, DAX, and Power Query, this dashboard demonstrates how business data can be transformed into meaningful insights within minutes. It is ideal for students, data analysts, managers, and business professionals looking to understand sales trends and improve decision-making.
